# 프로젝트 개요

## 📖 소개

**유튜브 다운로더 CLI**는 파이썬으로 개발된 명령줄 인터페이스 프로그램으로, 사용자가 터미널에서 간편하게 유튜브 동영상을 다운로드할 수 있도록 지원합니다.

---

## 🎯 프로젝트 목표

### 주요 목표
1. **사용자 친화적인 CLI 제공**: 직관적인 명령어와 옵션으로 누구나 쉽게 사용
2. **다양한 포맷 지원**: MP4, MP3 등 여러 형식으로 다운로드 가능
3. **고성능**: 빠르고 안정적인 다운로드 경험 제공
4. **확장 가능성**: 향후 기능 추가가 용이한 구조

### 부가 목표
- 재생목록 일괄 다운로드
- 진행률 실시간 표시
- 메타데이터 및 썸네일 저장
- 사용자 설정 관리

---

## ✨ 핵심 기능

### 1. 동영상 다운로드
- 유튜브 URL을 입력받아 동영상 다운로드
- 다양한 해상도 선택 (4K, 1080p, 720p, 480p 등)
- 비디오 포맷 선택 (MP4, WebM 등)

### 2. 오디오 다운로드
- 오디오만 추출하여 MP3로 저장
- 고품질 오디오 옵션 제공

### 3. 재생목록 다운로드
- 유튜브 재생목록 전체 다운로드
- 개별 동영상 선택 다운로드 옵션

### 4. 진행률 표시
- Rich 라이브러리를 활용한 시각적 진행률 바
- 다운로드 속도 및 남은 시간 표시

### 5. 메타데이터 관리
- 동영상 제목, 설명, 업로드 날짜 저장
- 썸네일 이미지 다운로드
- JSON 형식으로 메타데이터 저장

### 6. 설정 관리
- 기본 다운로드 경로 설정
- 선호하는 화질/포맷 저장
- 환경 변수 지원

---

## 🎨 사용 예시

### 기본 다운로드
```bash
ytdl download https://www.youtube.com/watch?v=dQw4w9WgXcQ
```

### 특정 화질로 다운로드
```bash
ytdl download https://www.youtube.com/watch?v=dQw4w9WgXcQ --quality 1080p
```

### 오디오만 다운로드
```bash
ytdl download https://www.youtube.com/watch?v=dQw4w9WgXcQ --audio-only
```

### 재생목록 다운로드
```bash
ytdl download https://www.youtube.com/playlist?list=PLxxxxxx --playlist
```

### 다운로드 경로 지정
```bash
ytdl download https://www.youtube.com/watch?v=dQw4w9WgXcQ -o ~/Videos/
```

## 🏗 아키텍처 개요

### 모듈 구조
```
youtube_downloader/
├── cli.py          # CLI 진입점 및 명령어 정의
├── downloader.py   # yt-dlp 래퍼 및 다운로드 로직
├── config.py       # 설정 관리 (Pydantic)
├── models.py       # 데이터 모델 정의
└── utils.py        # 유틸리티 함수
```

### 주요 컴포넌트

#### 1. CLI 레이어 (`cli.py`)
- Click을 사용한 명령어 파싱
- 사용자 입력 검증
- Rich를 활용한 UI 렌더링

#### 2. 다운로더 레이어 (`downloader.py`)
- yt-dlp 라이브러리 래핑
- 다운로드 옵션 설정
- 진행률 콜백 처리

#### 3. 설정 레이어 (`config.py`)
- Pydantic을 사용한 설정 검증
- 환경 변수 및 설정 파일 로드
- 기본값 관리

#### 4. 데이터 모델 (`models.py`)
- 동영상 정보 모델
- 다운로드 옵션 모델
- 메타데이터 모델

## 👥 대상 사용자

### 주 사용자
- 유튜브 콘텐츠를 오프라인에서 시청하고 싶은 사용자
- 터미널 환경을 선호하는 개발자
- 일괄 다운로드가 필요한 콘텐츠 크리에이터

### 사용 시나리오
1. **학습 자료 보관**: 강의 영상을 로컬에 저장
2. **콘텐츠 백업**: 자신의 유튜브 영상 백업
3. **오프라인 시청**: 인터넷 연결 없이 동영상 시청
4. **오디오 추출**: 음악이나 팟캐스트를 MP3로 변환
