# 웹 인터페이스 가이드

## 📋 개요

YouTube Downloader 웹 인터페이스는 브라우저에서 간편하게 YouTube 동영상을 다운로드할 수 있는 웹 애플리케이션입니다.

---

## 🚀 시작하기

### 서버 실행

```bash
# 개발 서버 실행
uv run uvicorn youtube_downloader.web.app:app --reload --port 8000
```

### 접속

브라우저에서 다음 URL을 열어주세요:
- **메인 페이지**: http://localhost:8000
- **API 문서**: http://localhost:8000/docs
- **WebSocket 테스트**: http://localhost:8000/static/test_ws.html

---

## 📖 사용 방법

### 1. URL 입력

YouTube 동영상 URL을 입력 필드에 붙여넣으세요.

**지원 형식**:
- `https://www.youtube.com/watch?v=VIDEO_ID`
- `https://youtu.be/VIDEO_ID`

URL을 입력하면 자동으로 유효성 검사가 진행됩니다.

### 2. 동영상 미리보기

유효한 URL을 입력하면 자동으로 동영상 정보가 표시됩니다:
- 썸네일
- 제목
- 채널명
- 길이

### 3. 옵션 선택

#### 화질 선택
- 최고 화질
- 1080p
- 720p (기본값)
- 480p
- 360p

#### 오디오 품질
- 320kbps
- 192kbps (기본값)
- 128kbps
- 96kbps
- 64kbps

#### 추가 옵션
- ☐ 오디오만 다운로드
- ☐ 메타데이터 저장
- ☐ 썸네일 저장

### 4. 다운로드 시작

"다운로드 시작" 버튼을 클릭하면:
1. 서버에 다운로드 요청 전송
2. WebSocket 연결 자동 생성
3. 실시간 진행률 표시 시작

### 5. 진행률 확인

다운로드 중에는 다음 정보가 실시간으로 업데이트됩니다:
- **진행률 바**: 0-100%
- **다운로드 크기**: 15.2 MB / 33.8 MB
- **속도**: 2.5 MB/s
- **남은 시간**: 00:15

### 6. 파일 다운로드

다운로드가 완료되면:
- 완료 메시지 표시
- "파일 다운로드" 버튼 활성화
- 버튼 클릭 시 파일 자동 다운로드

## 🎨 UI 기능

### 다크 모드

우측 상단의 🌙 버튼을 클릭하여 다크 모드를 전환할 수 있습니다.
- 라이트 모드: 밝은 배경
- 다크 모드: 어두운 배경 (눈의 피로 감소)

설정은 자동으로 저장되어 다음 방문 시에도 유지됩니다.

### 반응형 디자인

모든 화면 크기에서 최적화된 레이아웃을 제공합니다:
- **모바일** (<768px): 단일 컬럼, 풀 너비
- **태블릿** (768px-1024px): 2컬럼 옵션 레이아웃
- **데스크톱** (>1024px): 최대 너비 1024px

## 🔧 고급 기능

### 동시 다운로드

여러 동영상을 동시에 다운로드할 수 있습니다:
1. 첫 번째 다운로드 시작
2. 새 탭에서 같은 페이지 열기
3. 두 번째 다운로드 시작

각 다운로드는 독립적으로 진행됩니다.

### 취소 기능

다운로드 중 "취소" 버튼을 클릭하면:
- WebSocket 연결 종료
- 다운로드 중단
- 폼 초기화

## 🐛 문제 해결

### URL이 인식되지 않아요
- YouTube URL 형식이 올바른지 확인하세요
- 동영상이 삭제되거나 비공개가 아닌지 확인하세요

### 다운로드가 시작되지 않아요
- 서버가 실행 중인지 확인하세요
- 브라우저 콘솔에서 에러 메시지를 확인하세요
- 네트워크 연결을 확인하세요

### 진행률이 업데이트되지 않아요
- WebSocket 연결이 정상인지 확인하세요
- 브라우저를 새로고침해보세요
- 방화벽 설정을 확인하세요

### 파일 다운로드가 안 돼요
- 다운로드가 완료될 때까지 기다려주세요
- 브라우저의 다운로드 설정을 확인하세요
- 팝업 차단이 해제되어 있는지 확인하세요

## 🎯 팁

### 빠른 다운로드
- 화질을 낮추면 다운로드 속도가 빨라집니다
- 오디오만 다운로드하면 파일 크기가 작습니다

### 파일 크기 절약
- 오디오 품질을 낮추면 파일 크기가 줄어듭니다
- 64kbps: 4시간 영상 약 115MB
- 320kbps: 4시간 영상 약 576MB

### 배터리 절약 (모바일)
- 다크 모드를 사용하세요
- 다운로드 중에는 화면을 켜두지 않아도 됩니다

---

## 💡 알아두면 좋은 점

- 다운로드된 파일은 서버의 `downloads/` 디렉토리에 저장됩니다
- 완료된 작업은 1시간 후 자동으로 삭제됩니다 (선택사항)
- WebSocket 연결은 다운로드 완료 후 자동으로 종료됩니다
- 여러 브라우저 탭에서 동시에 사용할 수 있습니다
